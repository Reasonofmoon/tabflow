# Store Listing Draft — v2.0.0

## Extension Name

Moon-TabFlow — 탭 & 북마크 통합 매니저

## Short Description (45 chars max)

All-in-one tab & bookmark manager with AI sort.

## Detailed Description

### English

Moon-TabFlow is the all-in-one tab and bookmark manager that lives in Chrome's side panel. v2.0 brings powerful bookmark restoration and smart organization.

**Key Features:**

📋 **Tab Manager** — View and manage all tabs across every window in one place
📁 **Smart Grouping** — Auto-classify tabs into groups by keyword-based AI
💤 **Memory Saver** — Discard inactive tabs to free up RAM instantly
🔄 **Duplicate Cleaner** — Find and close duplicate tabs with one click
📊 **Bookmark Dashboard** — Health score, duplicate detection, broken link checker
🤖 **AI Auto-Classify** — Sort bookmarks into 20 smart categories automatically
📦 **Backup & Restore** — Full JSON backup with flat restore (no messy folders)
🗂️ **Folder Organizer** — Rename, sort, flatten, or delete empty bookmark folders
🃏 **Card View** — Browse bookmarks as visual cards with category tags
🎨 **Theme System** — Light, Dark, and System-matching modes
⌨️ **Keyboard Shortcuts** — Ctrl+Shift+K (search), Ctrl+Shift+D (discard)

**What's New in v2.0:**
- 🆕 Flat Restore — Imports all bookmarks as a flat list (no domain folders)
- 🆕 20 Categories — Expanded AI classification from 9 to 20 categories
- 🆕 Folder Organizer — Tree view with rename, sort, flatten, delete tools
- 🆕 Empty folder detection and cleanup
- 🆕 Robust backup import with progress tracking

**Bonus:**
- Drag-and-drop tab reordering
- Right-click context menu for quick actions
- Real-time updates when tabs change
- Zero external dependencies — 100% local processing

### 한국어

Moon-TabFlow는 크롬 사이드 패널에서 동작하는 올인원 탭 & 북마크 관리 도구입니다. v2.0에서 강력한 북마크 복원과 스마트 정리 기능이 추가되었습니다.

**주요 기능:**

📋 **탭 매니저** — 모든 윈도우의 탭을 한 화면에서 관리
📁 **스마트 그룹핑** — 키워드 AI 기반 자동 탭 분류
💤 **메모리 절약** — 비활성 탭 Discard로 RAM 즉시 확보
🔄 **중복 정리** — 중복 탭 원클릭 감지 & 닫기
📊 **북마크 대시보드** — 건강도 점수, 중복 감지, 깨진 링크 검사
🤖 **AI 자동 분류** — 20개 카테고리로 북마크 자동 정리
📦 **백업 & 복원** — JSON 백업 + 플랫 복원 (폴더 없이 깔끔하게)
🗂️ **폴더 정리 도구** — 이름 변경, 정렬, 평탄화, 빈 폴더 삭제
🃏 **카드 뷰** — 카드 형태로 북마크 시각적 탐색
🎨 **테마 시스템** — 라이트 / 다크 / 시스템 모드
⌨️ **단축키** — Ctrl+Shift+K (검색), Ctrl+Shift+D (Discard)

**v2.0 새 기능:**
- 🆕 플랫 복원 — 도메인 폴더 없이 모든 북마크를 평면 목록으로 복원
- 🆕 20개 카테고리 — AI 분류를 9개에서 20개로 확장
- 🆕 폴더 정리 도구 — 트리 뷰에서 이름 변경, 정렬, 평탄화, 삭제
- 🆕 빈 폴더 감지 및 일괄 정리
- 🆕 진행률 표시가 있는 안정적 백업 복원

**추가 기능:**
- 드래그앤드롭 탭 정렬
- 우클릭 컨텍스트 메뉴
- 실시간 탭 변경 감지
- 외부 의존성 없음 — 100% 로컬 처리

## Privacy Highlights

- No tracking, no analytics, no telemetry
- No external server connections
- All data stays in your browser (chrome.storage.local)
- AI classification uses keyword matching only — no API calls
- Bookmark backup/restore is 100% offline
