# Store Listing Draft

## Extension Name

Moon-TabFlow — 탭 & 북마크 통합 매니저

## Short Description (45 chars max)

Manage tabs & bookmarks in one side panel.

## Detailed Description

### English

Moon-TabFlow is the all-in-one tab and bookmark manager that lives in Chrome's side panel.

**Key Features:**

📋 **Tab Manager** — View and manage all tabs across every window in one place
📁 **Smart Grouping** — Auto-classify tabs into groups by keyword-based AI
💤 **Memory Saver** — Discard inactive tabs to free up RAM instantly
🔄 **Duplicate Cleaner** — Find and close duplicate tabs with one click
📊 **Bookmark Dashboard** — Health score, duplicate detection, broken link checker
🤖 **AI Classification** — Automatically sort bookmarks into 9 smart categories
🃏 **Card View** — Browse bookmarks as visual cards with category tags
🎨 **Theme System** — Light, Dark, and System-matching modes
⌨️ **Keyboard Shortcuts** — Ctrl+Shift+K (search), Ctrl+Shift+D (discard)

**Bonus:**
- Drag-and-drop tab reordering
- Right-click context menu for quick actions
- JSON backup and restore for bookmarks
- Real-time updates when tabs change
- Zero external dependencies — 100% local processing

### 한국어

Moon-TabFlow는 크롬 사이드 패널에서 동작하는 올인원 탭 & 북마크 관리 도구입니다.

**주요 기능:**

📋 **탭 매니저** — 모든 윈도우의 탭을 한 화면에서 관리
📁 **스마트 그룹핑** — 키워드 AI 기반 자동 탭 분류
💤 **메모리 절약** — 비활성 탭 Discard로 RAM 즉시 확보
🔄 **중복 정리** — 중복 탭 원클릭 감지 & 닫기
📊 **북마크 대시보드** — 건강도 점수, 중복 감지, 깨진 링크 검사
🤖 **AI 자동 분류** — 9개 카테고리로 북마크 자동 정리
🃏 **카드 뷰** — 카드 형태로 북마크 시각적 탐색
🎨 **테마 시스템** — 라이트 / 다크 / 시스템 모드
⌨️ **단축키** — Ctrl+Shift+K (검색), Ctrl+Shift+D (Discard)

**추가 기능:**
- 드래그앤드롭 탭 정렬
- 우클릭 컨텍스트 메뉴
- JSON 백업 & 복구
- 실시간 탭 변경 감지
- 외부 의존성 없음 — 100% 로컬 처리
